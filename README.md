# Bank Account OOP Project

A Java project demonstrating Object Oriented Programming concepts.

## Features
- Deposit and withdraw money
- Transfer between accounts
- Apply interest (Savings Account)
- Credit limit spending (Premium Account)
- Calculate fees per account type

## Concepts Used
- Encapsulation
- Inheritance
- Polymorphism (Overriding and Overloading)
- Abstraction (Abstract classes and Interfaces)
- ArrayLists

## How to Run
1. Clone the repository
2. Open in NetBeans
3. Run Main.java
